Программа для навигации по файловой системе и создания текстовых заметок в Windows.

Тренируется:
	- библиотека std::filesystem
	- работа с файловым и строковым потоками
	- выстраивание логики обработки команд
	- система оповещений об ошибках

Для корректного начала работы программы необходимо, чтобы при запуске в одной папке находились:
	- исполняемый файл
	- файл  '[log] mynotes_dir.txt

При первом запуске необходимо с помощью команды
-$ sndir <абсолютный_путь>
задать путь, по которому будут находиться заметки. Он будет сохранен в лог-файл и при повторных запусках автоматически считан и отображен в первой строке (mynotes_dir ...)

Путь текущей директории отображается во второй строке программы (current path ...). По файловой системе можно перемещаться командами.

Отобразить все доступные команды:
-$ show com
